# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/drive/1IGaPta0aDXF1eOBIrONDwJfG3LOVaYbK

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. (Optional) Create a `.env.local` file and set `VITE_GOOGLE_API_KEY=your_key` to enable the Gemini/Google GenAI assistant. If not provided, the assistant will use an offline fallback.
3. Run the app:
   `npm run dev`
